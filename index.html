    Projects

    Leave Management System using C# ASP.NET Core

    A C# driven project using the ASP.NET Core framework that allows a company to allocate and keep track of different leave types and requests for its employees. The project features both a basic Employee view and an Administrator view that adds administrative functions like adding/editing leave types or periods, as well as reviewing and accepting/denying leave requests.

    Node.js Chatbot for Twitch Streaming

    A chatbot that runs locally as a Node.js program. The bot can be programmed to respond to different commands that are entered into the Twitch streamer's chat. An asynchronous function is used to handle messages that are sent to OpenAI's ChatGPT, where ChatGPT's response is then sent to the chat as a message. Also features a Python module that can play a collection of audio files as sound commands using the Playsound library.

    Binary Search Tree using C++

    A project that uses a binary search tree data structure in order to sort course listings in order by their course number. Project also allows user to view individual course information, which will also list courses that are prerequisites for the course that they are viewing. Repository also contains a text document that details pseudocode for applying the same sorting method using vectors or a hash table, and rationale for why a binary search tree was chosen as the best option for the use case.

    Dashboard w/ CRUD Module

    Application utilizing Python, MongoDB, and CRUD functionality. Provides management of a collection using data set provided by Austin Animal Shelter. This project creates a dashboard for Grazioso Salvare, who wishes to identify dogs for training using specific parameters.

    Embedded C for Microcontroller

    Two .c files containing code meant to be run on a TI CC3220S-LAUNCHXL microcontroller. The uartecho.c file contains code that modifies the on/off state of the mcu's LED lights by utilizing a state machine that interprets user input. The gpiointerrupt.c file contains code that runs the mcu as a prototype thermometer, using the ambient room temperature to determine the heat mode (on/off) based on the setpoint that is set by the user.

    3D Scene using OpenGL

    A program that runs a 3D scene using WASD and mouse controls. The goal of the project was to replicate a reference photo containing four objects. Each 3D object was constructed solely using vertex arrays and matrix transformations.
